                            amateur italian translation / traduzione amatoriale italiana


### Description / Descrizione

**EN**:
This project was born out of love for Darya Noghani's video game, **Small Saga**, with the aim of creating an amateur translation into Italian.
The files within the project are the game's dialogue files. I extracted them from the package.nw file where they were contained, which I first renamed as .zip (package.zip) and then unzipped for editing.
I am open to suggestions for translations, and feel free to use this project as a basis for other future translations into other languages.

**IT**:
Questo progetto nasce per l'amore per il videogioco di Darya Noghani, **Small Saga**, con l'intento di creare una traduzione amatoriale in lingua italina.
I file all'interno del progetto sono i file di dialogo del gioco. Li ho estratti dal file package.nw dove erano contenuti, che ho prima rinominato come .zip (package.zip) e poi unzippato per editare.
Sono aperto a suggerimenti per le traduzioni e usate pure questo progetto come base per altre future traduzioni in altre lingue.


###  Instructions for translating game files / Istruzioni per tradurre i file di gioco

**EN**: To translate the game files, you need the package.nw file.

1) Go to the game folder (usually _<YOUR-PATH-TO-STEAM_LIBRARY>/steamapps/common/Small Saga_).
2) Rename the package.nw file to package.zip.
3) Extract the contents to a dedicated directory (I recommend 7-Zip).
4) The files to be edited are the .txt files, which contain all the dialogues and texts in the game.
5) (Optional) I suggest initializing a git repository on the dedicated directory to track your changes.
6) Alternatively, you can fork this project from its first commit and then package the modified files into your package.nw.
